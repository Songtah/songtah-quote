# 開發漏斗與客情追蹤 — 業務團隊操作說明

> 一頁看完。有問題找管理員。

---

## 🎯 開發漏斗:新客戶開發的進度板

**在哪裡**:系統上方選單「業務開發」→ 點「🎯 開發漏斗」分頁。

**這是什麼**:全台**新開業的牙醫診所**會自動出現在這裡(每月更新),加上我們手動加入的開發對象。每一家都有一個「階段」,代表開發進度:

```
線索 → 已接觸 → 試用中 → 報價中 → 已成交
                                  ↘ 流失
```

**你要做的三件事**:

1. **認領**:看到沒有負責人的診所,想跑就點「認領」——它就是你的了,別人看得到。
2. **推進**:每次拜訪後,用該診所右邊的下拉選單更新階段。去拜訪過了→「已接觸」;送了樣品→「試用中」;開了報價單→「報價中」;成交→「已成交」。**30 秒的事,當天做完**。
3. **注意紅字**:
   - 「⚠ 逾期」= 你之前約定的追蹤日已經過了,優先處理
   - 「⏳ 停滯 N 天」= 這家太久沒動靜,週會可能會被問到 😄

**為什麼值得做**:新開業診所前幾個月是採購設備和材料的黃金期,誰先去誰贏。這個板子就是讓你比別人早知道「哪裡開了新診所」。

## ✓ 待追蹤結案:把該回的客戶回完

**在哪裡**:儀表板首頁的「待追蹤客情」。

**改了什麼**:以前的待追蹤清單每個月 1 號會「自動消失」——其實沒消失,是系統只顯示當月的。現在**沒處理完的會一直留著**,跨月也在。

**你要做的**:

1. 寫客情紀錄時,有需要回頭處理的就照舊勾「需要追蹤」、填追蹤日。
2. 事情處理完之後,到待追蹤清單點該筆的「**✓ 結案**」——它就從清單消失。
3. 點錯了不用怕:請管理員到 Notion 把那筆的「追蹤已結案」取消勾選,就會回來。

**第一次打開會看到很多筆**:那是過去累積的(以前系統藏起來的),很多其實早就處理過了——花點時間清一輪,處理過的直接結案,之後清單就是乾淨的真實待辦。

## 常見問題

| 問題 | 答案 |
|---|---|
| 認領錯了怎麼辦 | 找管理員改負責業務 |
| 這家已經在跟了,但不在漏斗裡 | 找管理員把該客戶加入漏斗(設定開發階段) |
| 成交之後還要管嗎 | 推進到「已成交」就好,之後走正常客情維護 |
| 確定不做了的 | 選「流失」,不要放著變殭屍 |
